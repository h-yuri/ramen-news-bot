今日のハイライト
【コスト】

日本スーパー協会によると、8月まで店頭価格上昇が継続。ナフサ高により包装資材（ビニール袋・トレー）の仕入れが30％↑、ラップの仕入れ価格は2倍超に達している（旭化成供給問題も影響）。砂糖も中東情勢と円安で3～5％値上げ。6月には食料品1000品目超が値上げされ、ガス代も高騰し「五重苦」の状態。ラーメンの原価率は円安の影響で5割超に上昇しており、特にラード価格高騰により、とんこつラーメン店が深刻な影響を受けている。

【人件費】

飲食店責任者の7割が人件費高騰と人材不足に不安を感じている。2025年度上期には東京の飲食業界平均給与が月給30万円台へ上昇する見込みで、人材確保には最低賃金（佐賀県で956円）では不十分であり、時給1200円が理想とされている。2025年度の最低賃金改定を見越した具体的な対策が必須となっている。

【客数・消費】

9月の消費者物価指数は2.9％上昇、食料品値上がりが続く中、値上げによる客離れが回転ずしや焼肉チェーンで顕在化し、消費層の二極化が進んでいる。観光地の飲食店では中国人観光客が半減超など、外国人客が明らかに減少しており、「空洞化」リスクが指摘されている。2025年の外食産業は全店売上高7.3%増だが、これは客単価上昇によるもので、客数は頭打ち傾向。食料品の消費税ゼロが議論されると、外食からスーパーの惣菜などへ客が流れる懸念があり、外食業界は来店客離れに打撃を受けると反対を表明している。訪日客は4000万人突破も約7割の飲食店は取り込めておらず、集客には「旅マエ予約」が重要である。

【競合・業界】

2025年度のラーメン店倒産は物価高を主因として過去2番目に多い57件に達した。焼肉店倒産も2025年に46件と年間最多を更新しており、値上げ・高級化での客離れ、低価格チェーンとの競争激化が背景にある。町中華の「息切れ倒産」が増加している一方で、大手チェーンは値上げでも好調を維持しており、個店と大手で明暗が分かれている。飲食店全体で、コスト増の32.3%しか価格転嫁できていない現状があり、「値上げすれば客離れ、しなければ倒産」というジレンマに直面している。

【世界情勢→経営】

中東情勢の不安定化（イランがホルムズ海峡を通航する船舶に料金徴収を主張）や米利上げ観測は、原油価格高騰を招き、ナフサ高騰、ガス代・輸送コストの上昇、プラスチック包装資材（ラップの仕入れ価格2倍超、ビニール袋・トレーの仕入れ30%↑）の仕入れ価格上昇に直結する。これにより、ラーメン店のエネルギーコスト、テイクアウト容器、食材保存用ラップなどのコストに直接影響が出る。また、青森県での震度6強地震や九州北部での線状降水帯発生、沖縄の台風接近による航空便欠航は、該当地域の来店客減少だけでなく、東北新幹線運休や物流の遅延・混乱を引き起こし、特定地域からの食材（例：青森ニンニク、九州豚骨）の安定供給に影響を与える可能性がある。

今月中
