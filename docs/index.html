今日のハイライト
【コスト】

中東情勢悪化でナフサ高騰し、テイクアウト用ビニール袋・トレー等の包装資材の仕入れ価格が30%上昇しています（記事30, 31）。キャベツの仕入れ価格が30円アップ（記事34）、イチゴ「紅ほっぺ」は20円アップ（記事38）。4月1日から調味料、加工食品、酒類・飲料など4000品目超が値上げ済みで（記事37）、円安により円建て輸入物価は2024年7月以来の伸びを見せ（記事42）、ラーメン店の原価率が5割を超える事例も報告されています（記事73）。

【人件費】

2025年度の最低賃金改定により、人件費高騰が確実視されています（記事48）。飲食業界の平均給与は上昇傾向で、東京では月給30万円台へ到達（記事50）。人手不足対策として、アルバイトの理想時給は1200円とされており、最低賃金だけでは人材確保が困難な状況です（記事54）。飲食店責任者の7割が人件費高騰と人材不足に不安を感じています（記事52）。

【客数・消費】

食料品の消費税ゼロ（または1%）案が浮上しており、実現すれば外食からスーパーへの客離れリスクが懸念されています（記事56, 57, 60）。2025年の外食全店売上高は7.3%増も、客数は頭打ちであり（記事63）、単価上昇が売上に貢献している状況です。東京都心で今年初の真夏日を記録（記事22）し、猛暑による温かいラーメン需要の減少が予測されます。

【競合・業界】

2025年度のラーメン店倒産は57件で過去2番目に多く、物価高・人手不足が主要因とされています（記事66, 69）。飲食店の価格転嫁率は32.3%と低く、「値上げすれば客離れ、しなければ倒産」のジレンマに陥っています（記事68）。大手ラーメンチェーン『天下一品』が首都圏で閉店ラッシュに直面（記事74）しており、焼肉店（46件）やハンバーガー店も倒産が過去最多を更新（記事70, 72）。

【世界情勢→経営】

米高性能AIモデルの対策会議開催（記事13）は、将来的なAI活用による店舗DX化、省力化のトレンドを示唆しており、人手不足対策として注目されます。28日から大雨などの防災気象情報が変更（記事6）されるため、災害時の客足変動予測やスタッフの安全確保計画を見直す機会です。クマの出没（記事4, 5）は、観光地や自然豊かな地域の店舗で客足減少、地域住民の外出控えによる売上減リスクを高めます。JR中央線の運転見合わせ（記事20）のような交通インフラのトラブルは、その沿線店舗の客足に一時的に直接影響します。
